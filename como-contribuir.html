Guía de Contribución

¿Cómo Contribuir? Paso a Paso

Aprende a contribuir a este proyecto colaborativo y forma parte de nuestra comunidad de desarrolladores. Sigue estos simples pasos para agregar tu información al directorio.

1

Fork el Repositorio

Haz un fork de este repositorio a tu cuenta de GitHub haciendo clic en el botón "Fork" en la página del repositorio. 

# Clona tu fork
git clone https://github.com/TU_USUARIO/RepoColaborativo.git
cd RepoColaborativo
2

Agrega tus Datos

Edita el archivo contributors.js y agrega tu información al final del array. 

{
  name: "Tu Nombre Completo",
  nickname: "tu_github_username",
  github: "https://github.com/tu_username", // Requerido
  linkedin: "https://linkedin.com/in/tu-perfil", // Opcional
  description: "Una breve descripción sobre ti y tu especialidad", // Requerido
  image: "https://tu-url-de-imagen.com/imagen.jpg", // Opcional
  hobbies: ["Programación", "Gaming", "Música", "Deportes"] // Máximo 4
},

Campos Requeridos:

  • name: Tu nombre completo
  • nickname: Tu username de GitHub
  • github: URL completa de tu perfil de GitHub
  • description: Breve descripción sobre ti (máximo 150 caracteres)
  • hobbies: Array con tus hobbies (máximo 4)

Campos Opcionales:

  • linkedin: URL de tu perfil de LinkedIn
  • image: URL de tu imagen de perfil personalizada

Sobre las Imágenes de Perfil:

  • Si no proporcionas una imagen, se usará automáticamente tu avatar de GitHub
  • Si tu imagen personalizada falla al cargar, se mostrará tu avatar de GitHub como respaldo
  • Si no tienes GitHub, se mostrarán tus iniciales con un color único
  • Formatos recomendados: JPG, PNG, WebP
  • Tamaño recomendado: 200x200px o mayor (se ajustará automáticamente)
3

Crea una Rama y Haz Commit

Crea una nueva rama para tus cambios y haz commit con un mensaje descriptivo. 

# Crea una nueva rama
git checkout -b feat/add-tu-nickname

# Agrega los cambios
git add contributors.js

# Haz commit con el formato correcto
git commit -m "feat: add [tu_nickname] to contributors"

# Envía los cambios a tu fork
git push origin feat/add-tu-nickname
4

Crea un Pull Request

Ve a GitHub y crea un Pull Request desde tu fork al repositorio original.

Título del PR:

feat: add [tu_nickname] to contributors

Descripción sugerida:

Agrega una breve descripción de lo que estás agregando, por ejemplo:

"Agregando mi información al directorio de participantes del taller GitHub Campus Experts."

Sistema de Validación Automática

Validación de Formato

El sistema verifica que el formato de tu contribución sea correcto y que todos los campos requeridos estén presentes.

Auto-merge

Si todas las validaciones pasan, tu PR será automáticamente mergeado y aparecerás en la página principal.

Corrección de Errores

Si hay errores, recibirás comentarios específicos sobre qué corregir. Puedes hacer nuevos commits para solucionarlos.

Ejemplos de Contribución

Ejemplo Correcto 

{
  name: "Ana María González",
  nickname: "anagonzalez",
  github: "https://github.com/anagonzalez",
  linkedin: "https://linkedin.com/in/ana-gonzalez-dev",
  description: "Estudiante de Ingeniería de Sistemas especializada en desarrollo web",
  hobbies: ["Programación", "Lectura", "Yoga", "Fotografía"]
},

Errores Comunes

Falta la coma al final del objeto
URL de GitHub incompleta o incorrecta
Más de 4 hobbies en el array
Descripción demasiado larga (más de 150 caracteres)
Nickname duplicado (ya existe en el sistema)
